⚡ Conectar Firebase
1
Crea proyecto en Firebase
Visita console.firebase.google.com y crea un nuevo proyecto
2
Activa Authentication + Firestore
En tu proyecto, activa Authentication (Email/Password) y Cloud Firestore
3
Obtén tu configuración
Ve a Configuración del proyecto → Tus apps → Web app y copia el objeto firebaseConfig
4
Pega tu configuración aquí
Reglas Firestore recomendadas
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null
        && request.auth.uid == userId;
    }
  }
}
